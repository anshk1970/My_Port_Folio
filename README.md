🌐 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website built using **HTML5, CSS3, and JavaScript**. This portfolio showcases my skills, education, projects, and contact information while providing a smooth user experience with animations and responsive design.

🚀 Live Demo

Automatically closes the mobile menu after clicking a navigation link.

* Valid email format

Provides:

* Error highlighting
